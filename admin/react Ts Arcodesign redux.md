## react Ts Arcodesign redux

环境准备：node .git.和Arcocli

arcocli全局安装

```
npm i -g @arco-design/arcocli
arco -v //查看版本号
```

### 安装

```
cd class
arco init admin # admin(项目名)
```

### 启动

```
cd admin

npm run dev或者yarn start
```
